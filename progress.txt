### Learnings (codebase patterns)

- pytest-httpx 0.36 uses `url=re.compile(...)`, NOT `url__regex=...` (test_exchange.py)
- Schema-mismatch checks must run BEFORE df-empty checks (allocation_get.py)
- Diff candidates need key cols (company+cc+gl) present on BOTH files; mutate gl_account on both sides before bumping value (test_diff.py)
- Fallback chain count = 1 initial + N attempts = 8 mocks for 7-day fallback exhaustion
- Subagent delegation NOT triggered (user invoked /ralph /tdd in interactive mode with caveman); direct Write/Edit OK because soft warnings only
- Live API key in .env (EXCHANGE_RATE_API_KEY) — tests must monkeypatch + bypass config.load_env() to assert API_KEY_MISSING
