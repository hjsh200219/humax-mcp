## 핵심 발견

- 적요(C25)는 SAP에서 정상 추출됨 (60% 채움). LLM이 못 본 게 아니라 토큰 폭발로 잘림
- 배부율(C31-34)은 raw에 이미 들어있음 → Step 3 수기 편집의 배부율 부분은 실제로 raw 변경 작업
- CC 마스터 한국어 lookup이 raw 안에 있음 → 외부 사전 불필요

## 17 시트 목록

`예산+실적` / `통합계정` / `CC` / `▶피벗` / `배부판` / `총액` / `지급수수료` / `EVCS(BP)` / `EVCS(AC)` / `EVCS(AC)_인증대행료` / `▶분석용` / `본사` / `법인` / `계정별배부액` / `기준` / `환율` / `IT서비스비용`

**Why:** raw 직접 검증으로 적요 누락 가설 H1 기각 + 배부율/CC 마스터 위치 확정. 도구 7개의 컬럼 매핑이 이 발견 기반으로 설계됨.

**How to apply:** 신규 도구 spec 작성 시 `예산+실적` 시트 C 컬럼 기준 직접 참조. 적요 추출은 C25, 배부율 변경은 C31-34, CC 한국어명은 `CC` 시트 C4에서 lookup.
